Make Your Own Math Quiz

You’re hired! You’ve been selected as a guest proctor for the galaxy-renowned Galactic Mathematic Academic Classic.

Your first assignment is to write a math quiz for the upcoming competition. Of course, we can’t trust new proctors to write just any old math quiz, so we’ll provide some constraints to help you design the best possible quiz. Good luck!

The Rules

  • For the purposes of this quiz, a math question is a sequence of digits, “+”, and “-” which the students must evaluate. For instance, the answer to the question “12+3-4” is 11.
  • Once you write a good first question, you’ll be allowed to proceed to write a second question, and so on. You can change any question at any time.
