Guess Hint Puzzle

Solution: Periscope
Answer: BENT PIPES

Written by Lewis Chen, Jon Schneider, Anderson Wang

The answers to this round are:

  • BABY IMPS
  • BOROUGHS
  • HORRIBLE
  • MISO SOUP
  • ONE TO ONE
  • SPEED BAG
  • TODDLERS
  • TOP-RATED
  • UNPEOPLE

Upon opening this round, we are presented with three puzzles, which are also lit up by lasers in the periscope device. Once one of these puzzles is solved, a button with the corresponding answer appears below the device that can be pressed, which may change the set of lit puzzles, potentially unlocking new ones.

All unlocked puzzles are listed below the device, but there are only three that are lit up at any given point.

From the layout of the pipes, as well as the periscope theming, it might suggest that we are looking from the side of the device and we get the puzzles in our line of sight. Furthermore, periscopes operate by a series of mirrors, so it seems logical that each junction contains a mirror that can be rotated.

There are 9 puzzles, 9 junctions, and 9 spaces that are not on the right. Each of these spaces holds one puzzle; each puzzle answer button controls a single mirror. By playing around with the buttons, it is possible to get a mapping of answers to junctions, as follows:

Somewhat promisingly, if we look at the squares next to the red, green, and blue outputs of the device on the right edge, the letters are R, G, and B. Furthermore, we see that the other letters in the perimeter are all O’s, except for another R, G, and B. Author’s Notes

Perhaps the most interesting decision we had to make in this round was with the initial state—a very early version of the puzzle had one puzzle that blocked unlocking the meta, which was not ideal. To help with this, we wrote a script to check all possible starting states and enumerate the sets of puzzles needed to unlock any given puzzle in each one, which let us filter for starting states that had few “bottlenecks” (defined as when you need to solve a certain puzzle X to unlock puzzle Y). The final one we chose has two such bottlenecks: you need to solve Mixed Message to unlock Thrifty/Thrifty, and you need to solve Mr. Worldwide to unlock Divide and Conquer, but otherwise no puzzle is required to unlock any other puzzle.

Like all the metas, the script was written in 2019 (originally by Lewis, who came up with the overall meta idea), so when we came back to it in 2021 we had no clue how to interpret its output. Eventually, Rahul and I (Anderson) sucked it up and spent the good portion of an evening reverse engineering the Python code, with Lewis contributing what he could remember, to figure out how it worked. In the process we ended up finding a small bug in the script which caused its output to be incorrect. If you’re up for a (very hard) challenge, here’s the script; see if you can find the bug! Hint: it’s a 1 character mistake.
