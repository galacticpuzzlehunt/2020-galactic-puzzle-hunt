Solution: #UnscienceAnAnimal
Answer: BEAUTY

Written by Seth Mulhall and Abigail Caron

Upon opening this puzzle, we are presented with a number of images of animals with nonsensical labels. Each image also has a large number in a corner.

The labels on each animal indicate the elements of a set with a canonical ordering. Each set is missing two elements. However, one of the two missing elements can be substituted with the animal in the picture.

The elements replaced by the animals are uniquely in positions 1 through 6 in their sets. If we index into the omitted element using the number given on the picture and order by the element replaced by the animal, we get BEAUTY, the answer to this puzzle.

Author’s Notes

This tweet was the inspiration for the puzzle, which was originally simply titled “what”. Hallucigenia was originally used as part of a matching step before extraction, but it proved to be unnecessarily difficult. This image will simply live on in these notes and our hearts.

Unfortunately, the given order of the images (alphabetical by common name), gives the "answer" of BUY TEA which is sort of compelling. I ended up with so much tea on the first day of hunt.
