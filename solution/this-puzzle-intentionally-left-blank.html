Solution: This Puzzle Intentionally Left Blank
Answer: EMPTY SET

Written by Alex Pei

Dumping the puzzle unicode codes as text reveals a large amount of zero width spaces broken up by different characters at the start of the puzzle. They come in run lengths 0–26, and by treating 0s as spaces, A=1, …, we can find the message "VIEW PUZZLE IN ARIAL UNICODE MS SINGLE SPACED THEN ORDER SUBPUZZLE ANSWERS HAIR EN SEPARATOR IDEOGRAPHIC FIGURE PUNCTUATION".

The next step is to download the required font by looking it up on the internet, followed by actually viewing the puzzle per the message. One option is to use a word processor such as LibreOffice, and coloring each character a different color with clever use of find and replace. Other ways involve changing the CSS in the element inspector, and setting all the values there. Either way, we should get an rectangular image, telling us we are on the correct path.

Now we can solve each subpuzzle as follows.

Location Subpuzzle Mechanism Answer
Center Hair/Thin There's 25 hair spaces and 1 thin space in each row. Take the position of the thin space in the alphabet to get a letter. REARRANGE
Bottom Center En There are 8 En spaces/quads in a row. This forms ASCII values, which can be read left to right, top to bottom. BLOCKS
Right Separators At the end of some lines, there are line separators and paragraph separators. Reading these types of lines as breaks using a Bacon cipher gives the answer. INTO
Bottom Left Ideographic This puzzle "answer" is just the shape these spaces depicts.
Top Right Figure/Math Reading figure spaces as digits, treating 2 adjacent digits as 1 number, and using the Medium Mathematical spaces as the operation they depict, we get answers between 1–26.
11×2 = 22 = V, 13−4 = 9 = I, 25/5 = 5 = E, 11+12 = 23 = W		 VIEW
Top Left Punctuation Reading each column of punctuation spaces as dots/dashes in Morse code gives us 1 letter per column IMPOSTERS

We can now view the bottom row of each puzzle, and find that there's a different sequence of spaces for each side, where each space appears in one of the subpuzzles above. Using these as indices into the subpuzzle answers above, we can get the left and right answers SNOWSTORMS and PELTMONGERS.

To get the answer for the puzzle itself though, we need to look at the cluephrase REARRANGE BLOCKS INTO shape VIEW IMPOSTERS. This is an instruction telling us that we can rearrange the subpuzzles into the shape of a square with a hole, of which there is only one way to do so. Finally, to view the IMPOSTERS, note that there are still 2 unused filler characters, the Hangul Filler and the Halfwidth Hangul Filler. Neither of these characters are actually classified as whitespace, and so by viewing these two characters and reading around the assembled shape, we get the answer EMPTY SET.

Author’s Notes

The line/paragraph separator characters subpuzzle was supposed to originally be unix vs windows line endings, but it turns out browsers and OSs really don't like to preserve these characters consistently, so I had to change this to other line break characters. Unfortunately on Windows, these characters show up as visible glyphs on Chrome's fallback font, but I couldn't find another line ending which wasn't visible/have weird behavior, so I left it as is.

Also, if you think really hard, you can imagine the rearranged shape is the face of an imposter.
